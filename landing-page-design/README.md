Навык-дизайнер посадочных страниц: агент собирает конвертящий лендинг по формуле
(hero → оффер → выгоды → соцдоказательство → FAQ → CTA), пишет заголовки и кнопки
по психологии конверсии и даёт копи-каркас под твой продукт. Не «красиво», а конвертит.

- **`SKILL.md`** — сам навык (англ. оригинал, ставится агенту).
- **`MARKETPLACE.md`** — подробно по-русски: боль, как работает, когда включать.
- **`manifest.yaml`** — метаданные карточки.

## Установка

Скопируй папку в `~/.claude/skills/landing-page-design/` — или возьми готовый промпт установки на
**[странице навыка](https://comandos.ai/hub/skills/landing-page-design)**. Триггеры: «сделай лендинг
для моего продукта», «спроектируй посадочную страницу», «какой должен быть hero на главной».

# 🔄 workflow-automation

**Проектирует надёжную автоматизацию процессов, которая не падает на полпути и возобновляется с места сбоя.**

Навык-архитектор автоматизации: говоришь «автоматизируй процесс X» (заявка → CRM → письмо) —
агент подбирает платформу (n8n, Temporal, Inngest) и собирает durable-схему с идемпотентностью,
ретраями и таймаутами. Рабочий процесс для прода вместо хрупкого скрипта, который умрёт от
первого сбоя сети.

- **`SKILL.md`** — сам навык (англ. оригинал, ставится агенту).
- **`MARKETPLACE.md`** — подробно по-русски: боль, как работает, когда включать.
- **`manifest.yaml`** — метаданные карточки.

## Установка

Скопируй папку в `~/.claude/skills/workflow-automation/` — или возьми готовый промпт установки на
**[странице навыка](https://comandos.ai/hub/skills/workflow-automation)**. Триггеры: «автоматизируй
процесс», «собери воркфлоу», «заявка с сайта в CRM».

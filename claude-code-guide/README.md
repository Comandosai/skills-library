# 📘 claude-code-guide

**Справочник-настройка, чтобы выжимать из Claude Code максимум — конфиги, «Thinking»-режимы, отладка.**

Не «сделай за меня», а «настрой и подскажи правильно». Навык превращает Claude Code в
эксперта по самому себе: помогает собрать грамотный `CLAUDE.md`, даёт словарь
«Thinking»-ключей для глубокого мышления, чек-лист отладки и лучшие практики.

Включаешь — и агент работает по уму: реже промахивается, реже требует переделок, делает
больше с первого захода. Подходит даже если ты не глубокий технарь.

- **`SKILL.md`** — сам навык (англ. оригинал, ставится агенту).
- **`MARKETPLACE.md`** — подробно по-русски: боль, как работает, когда включать.
- **`manifest.yaml`** — метаданные карточки.

## Установка

Скопируй папку в `~/.claude/skills/claude-code-guide/` — или возьми готовый промпт установки на
**[странице навыка](https://comandos.ai/hub/skills/claude-code-guide)**. Триггеры: «как настроить
Claude Code», «помоги составить CLAUDE.md», «агент застрял, что делать».

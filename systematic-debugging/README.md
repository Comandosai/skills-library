# 🐞 systematic-debugging

**При баге агент не лепит фиксы наугад, а находит настоящую причину — и только потом чинит.**

Навык-дисциплина для отладки: железное правило — никаких фиксов без расследования причины.
Агент идёт по четырём фазам (воспроизвести → сравнить → проверить гипотезу → починить корень),
трассирует данные от симптома к источнику и чинит причину одним точным фиксом с тестом.
Ты говоришь «у меня ошибка» — получаешь реально найденный и устранённый баг, а не «вроде поправил».

- **`SKILL.md`** — сам навык (англ. оригинал, ставится агенту).
- **`MARKETPLACE.md`** — подробно по-русски: боль, как работает, когда включать.
- **`manifest.yaml`** — метаданные карточки.

## Установка

Скопируй папку в `~/.claude/skills/systematic-debugging/` — или возьми готовый промпт установки на
**[странице навыка](https://comandos.ai/hub/skills/systematic-debugging)**. Триггеры: «у меня ошибка»,
«почему это не работает», «падает тест», «найди причину бага».

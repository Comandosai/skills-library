# ⚡ parallel-agents

**Разбивает большую задачу на независимые под-агенты и гонит их параллельно — а потом сводит результат.**

[![Telegram](https://img.shields.io/badge/Telegram-@ai__comandos-2CA5E0?style=for-the-badge&logo=telegram&logoColor=white)](https://t.me/ai_comandos)
[![Открыть на comandos.ai](https://img.shields.io/badge/Открыть_на_comandos.ai-→-A3E635?style=for-the-badge&logoColor=black)](https://comandos.ai/hub/skills/parallel-agents)

</div>

Навык-дирижёр для крупных задач: агент делит работу на независимые куски, запускает несколько
специализированных под-агентов одновременно (каждый со своей экспертизой и чистым контекстом),
а потом сводит всё в один отчёт с приоритетами. Быстрее на объёме, глубже за счёт разных точек зрения.

- **`SKILL.md`** — сам навык (англ. оригинал, ставится агенту).
- **`MARKETPLACE.md`** — подробно по-русски: боль, как работает, когда включать.
- **`manifest.yaml`** — метаданные карточки.

## Установка

Скопируй папку в `~/.claude/skills/parallel-agents/` — или возьми готовый промпт установки на
**[странице навыка](https://comandos.ai/hub/skills/parallel-agents)**. Триггеры: «проанализируй
весь проект разом», «сделай эти задачи параллельно», «запусти несколько агентов».

# 🔌 mcp-builder

**Строит MCP-серверы — мостики, через которые ИИ дёргает твой CRM, API и любые внешние сервисы.**

Навык-инженер интеграций: ты говоришь «подключи агента к моему CRM» — а он сам исследует API,
проектирует инструменты, пишет рабочий MCP-сервер на Python (FastMCP) или Node/TypeScript
(MCP SDK), собирает и тестирует через MCP Inspector. Один сервер понимают сразу несколько
клиентов: Claude, Cursor, Codex, Gemini.

- **`SKILL.md`** — сам навык (англ. оригинал, ставится агенту).
- **`MARKETPLACE.md`** — подробно по-русски: боль, как работает, когда включать.
- **`manifest.yaml`** — метаданные карточки.

## Установка

Скопируй папку в `~/.claude/skills/mcp-builder/` — или возьми готовый промпт установки на
**[странице навыка](https://comandos.ai/hub/skills/mcp-builder)**. Триггеры: «подключи агента
к моему CRM», «сделай интеграцию с сервисом», «построй MCP-сервер».

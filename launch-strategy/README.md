# 🚀 launch-strategy

**Строит пошаговый план запуска продукта — до, во время и после, по фазам и каналам.**

[![Telegram](https://img.shields.io/badge/Telegram-@ai__comandos-2CA5E0?style=for-the-badge&logo=telegram&logoColor=white)](https://t.me/ai_comandos)
[![Открыть на comandos.ai](https://img.shields.io/badge/Открыть_на_comandos.ai-→-A3E635?style=for-the-badge&logoColor=black)](https://comandos.ai/hub/skills/launch-strategy)

</div>

Навык-стратег для запуска продукта или фичи: агент собирает дорожную карту из пяти фаз
(от внутреннего теста до полного релиза), раскладывает каналы по фреймворку ORB
(свои → арендованные → заимствованные), готовит запуск на Product Hunt и выдаёт чек-листы
до/во время/после. Ты говоришь «как запустить мой продукт» — получаешь конкретный план,
а не общие советы.

- **`SKILL.md`** — сам навык (англ. оригинал, ставится агенту).
- **`MARKETPLACE.md`** — подробно по-русски: боль, как работает, когда включать.
- **`manifest.yaml`** — метаданные карточки.

## Установка

Скопируй папку в `~/.claude/skills/launch-strategy/` — или возьми готовый промпт установки на
**[странице навыка](https://comandos.ai/hub/skills/launch-strategy)**. Триггеры: «как запустить
мой продукт», «составь план запуска», «готовлюсь к Product Hunt».

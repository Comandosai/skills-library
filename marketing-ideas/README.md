Навык-стратег с библиотекой из ~140 проверенных маркетинг- и growth-ходов: агент уточняет
твой продукт, аудиторию и стадию → подбирает 3–5 релевантных приёмов → раскладывает их по
приоритету (быстрые победы → среднесрок → долгая игра). Конкретика вместо банальщины «веди блог».

- **`SKILL.md`** — сам навык (англ. оригинал, ставится агенту).
- **`MARKETPLACE.md`** — подробно по-русски: боль, как работает, когда включать.
- **`manifest.yaml`** — метаданные карточки.

## Установка

Скопируй папку в `~/.claude/skills/marketing-ideas/` — или возьми готовый промпт установки на
**[странице навыка](https://comandos.ai/hub/skills/marketing-ideas)**. Триггеры: «как продвигать
мой продукт», «дай маркетинговые идеи», «как привлечь клиентов».

Навык-аналитик: агент сначала спрашивает, какие решения ты принимаешь на основе данных,
а потом строит план трекинга (события → свойства → конверсии), настраивает GA4 / GTM /
Plausible с готовым кодом и даёт чек-лист валидации. Чистые данные под решения вместо
сотни бесполезных событий.

- **`SKILL.md`** — сам навык (англ. оригинал, ставится агенту).
- **`MARKETPLACE.md`** — подробно по-русски: боль, как работает, когда включать.
- **`manifest.yaml`** — метаданные карточки.

## Установка

Скопируй папку в `~/.claude/skills/analytics-tracking/` — или возьми готовый промпт установки на
**[странице навыка](https://comandos.ai/hub/skills/analytics-tracking)**. Триггеры: «настрой
аналитику», «что мне мерить», «составь план трекинга».

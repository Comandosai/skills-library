# 🔁 subagent-driven-development

**Выполняй план задачами — на каждую свежий субагент и двойная проверка: по ТЗ, затем по качеству.**

Навык-конвейер для исполнения готового плана: на каждую задачу — свежий субагент с чистым
контекстом, который реализует, тестирует и коммитит, а результат проходит двухступенчатое ревью
(сначала сверка с ТЗ, потом качество кода). Всё в одной сессии, без передачи человеку между задачами.

- **`SKILL.md`** — сам навык (англ. оригинал, ставится агенту).
- **`implementer-prompt.md` / `spec-reviewer-prompt.md` / `code-quality-reviewer-prompt.md`** — промпты субагентов.
- **`MARKETPLACE.md`** — подробно по-русски.
- **`manifest.yaml`** — метаданные карточки.

## Установка

Скопируй папку в `~/.claude/skills/subagent-driven-development/` — или возьми готовый промпт установки на
**[странице навыка](https://comandos.ai/hub/skills/subagent-driven-development)**.

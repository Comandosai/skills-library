Навык-аудитор для органического трафика: агент разбирает сайт по слоям
(техничка → скорость и Core Web Vitals → мета и контент → внутренние ссылки → E-E-A-T)
и сортирует находки по влиянию на ранжирование. Конкретные фиксы с приоритетами
вместо общих советов «пишите качественный контент».

- **`SKILL.md`** — сам навык (англ. оригинал, ставится агенту).
- **`MARKETPLACE.md`** — подробно по-русски: боль, как работает, когда включать.
- **`manifest.yaml`** — метаданные карточки.

## Установка

Скопируй папку в `~/.claude/skills/seo-audit/` — или возьми готовый промпт установки на
**[странице навыка](https://comandos.ai/hub/skills/seo-audit)**. Триггеры: «проверь сайт по
SEO», «сделай SEO-аудит», «почему сайт не ранжируется».

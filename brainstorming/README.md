Навык-мышление перед любой стройкой: агент сначала проектирует вместе с тобой
(вопросы по одному → 2–3 подхода с трейд-оффами → дизайн по секциям с проверкой),
а потом строит. Фундамент вместо горы кода, который жалко выбрасывать.

- **`SKILL.md`** — сам навык (англ. оригинал, ставится агенту).
- **`MARKETPLACE.md`** — подробно по-русски: боль, как работает, когда включать.
- **`manifest.yaml`** — метаданные карточки.

## Установка

Скопируй папку в `~/.claude/skills/brainstorming/` — или возьми готовый промпт установки на
**[странице навыка](https://comandos.ai/hub/skills/brainstorming)**. Триггеры: «давай устроим
мозговой штурм», «помоги придумать», «с чего начать проект».

# ✍️ copywriting

**Пишет продающий текст по брифу — лендинги, прайсинги, заголовки, CTA.**

Навык-копирайтер: агент сначала уточняет бриф (кому, зачем, какая боль, чем
отличаешься), а потом выдаёт текст страницы по секциям — заголовок, выгоды,
доказательства, обработка возражений, CTA — с вариантами заголовков и призывов
на выбор. Конкретика и язык клиента вместо воды и бузвордов.

- **`SKILL.md`** — сам навык (англ. оригинал, ставится агенту).
- **`MARKETPLACE.md`** — подробно по-русски: боль, как работает, когда включать.
- **`manifest.yaml`** — метаданные карточки.

## Установка

Скопируй папку в `~/.claude/skills/copywriting/` — или возьми готовый промпт установки на
**[странице навыка](https://comandos.ai/hub/skills/copywriting)**. Триггеры: «напиши текст
для лендинга», «перепиши прайсинг», «помоги с заголовком».

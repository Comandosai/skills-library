**Несколько независимых задач — раздать рою агентов и решить разом, а не по очереди.**

Навык-диспетчер: когда висит несколько не связанных между собой задач, он учит агента раздать
по одному субагенту на каждую и запустить их одновременно — но только после проверки, что задачи
правда независимы. В конце результаты сводятся и проверяются на конфликты.

- **`SKILL.md`** — сам навык (англ. оригинал, ставится агенту).
- **`MARKETPLACE.md`** — подробно по-русски.
- **`manifest.yaml`** — метаданные карточки.

## Установка

Скопируй папку в `~/.claude/skills/dispatching-parallel-agents/` — или возьми готовый промпт установки на
**[странице навыка](https://comandos.ai/hub/skills/dispatching-parallel-agents)**.

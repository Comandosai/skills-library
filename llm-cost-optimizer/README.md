# 💸 llm-cost-optimizer

**Режет расходы на LLM-API в 5–10 раз — маршрутизация, кэширование, инструментация.**

Проактивный советник по оптимизации расходов на LLM. Превращает расходы из чёрного ящика
в управляемый инженерный процесс: проводит cost audit по фичам, классифицирует запросы по сложности
для маршрутизации на дешёвые модели (80% запросов не нуждаются в топ-моделях), проектирует
кэширование, дизайнит мониторинг. На выходе — per-feature spend breakdown, scorecard оптимизации 0–100
и архитектура с прогнозом экономии до 80% от текущего бюджета.

- **`SKILL.md`** — сам навык (англ. оригинал, ставится агенту).
- **`MARKETPLACE.md`** — подробно по-русски.
- **`manifest.yaml`** — метаданные карточки.

## Установка

Скопируй папку в `~/.claude/skills/llm-cost-optimizer/` — или возьми готовый промпт установки на
**[странице навыка](https://comandos.ai/hub/skills/llm-cost-optimizer)**. Триггеры: «мои расходы на API слишком большие»,
«оптимизируй расходы на LLM», «какую модель выбрать для этой фичи», «запускаем AI-фичу».

# 🎯 prompt-engineering

**Пишет и чинит промпты за тебя — скажи, что нужно, получи сильный промпт со стабильным результатом.**

[![Telegram](https://img.shields.io/badge/Telegram-@ai__comandos-2CA5E0?style=for-the-badge&logo=telegram&logoColor=white)](https://t.me/ai_comandos)
[![Открыть на comandos.ai](https://img.shields.io/badge/Открыть_на_comandos.ai-→-A3E635?style=for-the-badge&logoColor=black)](https://comandos.ai/hub/skills/prompt-engineering)

</div>

Навык-промптолог внутри агента: говоришь «напиши промпт, чтобы нейросеть делала Y» или
«модель тупит — поправь» — и получаешь готовую конструкцию по проверенным паттернам
(few-shot, chain-of-thought, system-промпты, шаблоны). Стабильный результат вместо лотереи.
Работает с любой моделью и любым агентом.

- **`SKILL.md`** — сам навык (англ. оригинал, ставится агенту).
- **`MARKETPLACE.md`** — подробно по-русски: боль, как работает, когда включать.
- **`manifest.yaml`** — метаданные карточки.

## Установка

Скопируй папку в `~/.claude/skills/prompt-engineering/` — или возьми готовый промпт установки на
**[странице навыка](https://comandos.ai/hub/skills/prompt-engineering)**. Триггеры: «напиши
промпт чтобы нейросеть делала…», «модель тупит — поправь промпт», «составь системный промпт для бота».

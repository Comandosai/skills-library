Навык-инженер RAG: ты говоришь «сделай бота, который отвечает по моей базе» — и агент проходит
весь путь (умная нарезка по смыслу → эмбеддинги → векторная база → гибридный поиск с
переранжированием). На выходе — ответы на фактах из твоих данных, а не правдоподобные выдумки.
Ровно так устроен бот-консьерж COMANDOS.

- **`SKILL.md`** — сам навык (англ. оригинал, ставится агенту).
- **`MARKETPLACE.md`** — подробно по-русски: боль, как работает, когда включать.
- **`manifest.yaml`** — метаданные карточки.

## Установка

Скопируй папку в `~/.claude/skills/rag-implementation/` — или возьми готовый промпт установки на
**[странице навыка](https://comandos.ai/hub/skills/rag-implementation)**. Триггеры: «сделай бота,
который отвечает по моим документам», «собери поиск по базе знаний», «подключи RAG к проекту».

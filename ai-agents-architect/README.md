# 🧠 ai-agents-architect

**Проектирует архитектуру AI-агентов и мультиагентных систем под твою задачу — роли, оркестрацию, память, инструменты.**

Навык-архитектор для тех, кто строит не один промпт, а команду цифровых сотрудников: агент
сначала проектирует систему (роли → паттерн оркестрации: supervisor / swarm / plan-and-execute →
память → инструменты → режимы отказа), а потом строит. Продуманный дизайн вместо чёрного
ящика, который зацикливается на первом реальном кейсе.

- **`SKILL.md`** — сам навык (англ. оригинал, ставится агенту).
- **`MARKETPLACE.md`** — подробно по-русски: боль, как работает, когда включать.
- **`manifest.yaml`** — метаданные карточки.

## Установка

Скопируй папку в `~/.claude/skills/ai-agents-architect/` — или возьми готовый промпт установки на
**[странице навыка](https://comandos.ai/hub/skills/ai-agents-architect)**. Триггеры: «спроектируй
систему AI-агентов», «архитектура мультиагентной системы», «распредели роли между агентами».

# ✉️ email-sequence

**Пишет цепочки писем под задачу — онбординг, прогрев, реактивацию, продажу — с таймингом и логикой каждого письма.**

Навык-email-маркетолог: ты говоришь «сделай прогрев» или «нужна онбординг-серия писем» —
агент собирает готовую последовательность. Число писем, интервалы, у каждого свой смысл,
тема, превью-текст и тело. Система вместо разрозненных рассылок.

- **`SKILL.md`** — сам навык (англ. оригинал, ставится агенту).
- **`MARKETPLACE.md`** — подробно по-русски: боль, как работает, когда включать.
- **`manifest.yaml`** — метаданные карточки.

## Установка

Скопируй папку в `~/.claude/skills/email-sequence/` — или возьми готовый промпт установки на
**[странице навыка](https://comandos.ai/hub/skills/email-sequence)**. Триггеры: «сделай прогрев»,
«напиши онбординг-серию писем», «реактивировать уснувших подписчиков».
